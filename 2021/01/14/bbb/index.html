过程

调试环境

安装NodeJS
安装git
在git bash 中查看版本是否正确

  • git version
  • node -v
  • npm -v

安装hexo(npm环境要正确)

  • 在git bash 中输入：npm install -g hexo-cli

创建新文件夹myBlog

在 git bash中输入：hexo init myBlog
在 git bash中输入：cd myBlog （进入myBlog文件夹）
在 git bash中输入：npm install

预览效果

在 git bash中输入：hexo s

在http://localhost:4000中查看效果

在github中搭建仓库

配置 SSH key

  • 在 git bash中输入：git config –global user.name “用户名”
  • 在 git bash中输入：git config –global user.email “邮箱地址”
  • 在 git bash中输入：ssh-keygen -t rsa -C ‘上面的邮箱’
  • 按照提示完成三次回车，即可生成 ssh key。通过查看 ~/.ssh/id_rsa.pub 文件内容， + 获取到你的 SSH key
  • 登录 Github 上添加刚刚生成的SSH key

部署到github

调整_config.yml 配置文件配置参数

  • 将 type: ‘’ 改为

  • type: git
    repo: https://github.com/用户名/用户名.github.io.git
    branch: master

  • 安装部署插件hexo-deployer-git

  • 在 git bash中输入：npm install hexo-deployer-git –save

  • 部署上传

  • 在 git bash中输入：hexo g -d
